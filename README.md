# Quiz GitHub Actions

Este repositorio es para practicar **GitHub Actions** paso a paso.  
Es un proyecto sencillo con una calculadora en JavaScript y varias tareas de CI.

La idea no es hacer algo grande, sino aprender cómo se configuran los workflows.

---

## ¿Qué hay en este repo?

- Un pequeño proyecto de Node.js:
  - `calculadora.js`
  - `test-calculadora.js`
  - `package.json`
- Carpeta de workflows:
  - `.github/workflows/ejercicio1.yml`
  - `.github/workflows/ejercicio2.yml`
  - `.github/workflows/ejercicio3.yml`
  - `.github/workflows/ejercicio4.yml`
  - `.github/workflows/ejercicio5.yml`

Cada archivo `ejercicioX.yml` es un workflow diferente para el quiz.

---

## Requisitos

Para probar el proyecto en tu máquina necesitas:

- **Git**
- **Node.js** (versión 16 o 18 está bien)
- **npm**
